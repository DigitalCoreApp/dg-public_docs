# Importing Data from Excel or CSV

Use the import tool to load engagements, plan values, or actual values from a file. Useful for the first month and for catching up after a period.

## What you can import

- Engagements and contracts.
- Finance plan values.
- Finance actuals.
- Performance actuals.
- Capacity hours.

## Step 1: Pick a template

Go to **Configure**, then **Import**. Pick the type of import you want and click **Download template**. The template includes the columns required.

## Step 2: Fill in your file

Open the file in Excel or Google Sheets. One row per record.

- Use the date format shown in the header.
- Leave the ID column blank for new records.
- Use exact engagement names. Spelling has to match.

## Step 3: Upload

Drag the file into the upload box. DigitalCore reads it and shows a preview.

- Green rows are ready to import.
- Amber rows need a small fix (for example, an unknown engagement name).
- Red rows are blocked.

## Step 4: Confirm

Click **Import**. The page shows progress and a summary when done. You can download an error log for any row that did not import.

## Tips

- Run a small test import first (5 to 10 rows).
- Keep your import files in a shared folder so you can re-run them if needed.
- For monthly actuals, consider building a check-in instead of importing each month.
