# Setting Up Templates

Templates define what you track on each engagement. Build them once at the organisation level, then attach them to engagements.

## The three template types

- **Finance.** Revenue and cost lines (P&L items).
- **Performance.** KPIs (response time, uptime, satisfaction).
- **Capacity.** Roles and target hours.

## Step 1: Build a template

1. Go to **Configure**, then **Templates**.
2. Pick the type (Finance, Performance, Capacity).
3. Click **New Template** and give it a name.
4. Add items by picking from the library.

## Step 2: Set defaults per item

For each item:

- **Finance.** P&L category, currency, plan default.
- **Performance.** KPI, target, threshold operator.
- **Capacity.** Role, target hours per period.

## Step 3: Assign to an engagement

1. Open the engagement.
2. Go to **Templates**.
3. Pick one template per type.
4. Set start and end dates.

The plan grid for the engagement uses these templates.

## Reusing templates

A template can be assigned to many engagements. Update the template once and engagements that use it pick up the change for future periods.

## Tips

- Start with the default templates DigitalCore ships and adjust.
- Keep template names short and clear ("Managed Services - Standard", not "MS Std v3 final").
- Use the library to keep item names consistent across templates.
