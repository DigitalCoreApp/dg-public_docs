# Excel and CSV Import

Use the import tool for first-time loads or to catch up on past periods.

## What you can import

- Engagements and contracts.
- Finance plan values and actuals.
- Performance actuals.
- Capacity hours.

## How it works

1. Go to **Configure**, then **Import**.
2. Pick the type and download the template.
3. Fill in the file. One row per record.
4. Drag the file into the upload box.
5. Review the preview: green rows are ready, amber rows need a fix, red rows are blocked.
6. Click **Import**.

DigitalCore shows a summary when done. You can download an error log for any row that did not import.

## Common issues

- **Engagement name.** Must match exactly. Watch for trailing spaces.
- **Date format.** Use the format shown in the template header (YYYY-MM-DD).
- **Currency.** Numbers, no symbols. Decimals with a point.

## For ongoing data

Once your engagements are set up, prefer monthly check-ins to imports. They are quicker and link straight into the activity log.
