# Connections and Integrations

DigitalCore connects with the tools you already use. This section covers the four main types of connection.

## AI assistant connection (MCP)

MCP is the standard DigitalCore uses to expose tools to AI assistants like Claude or ChatGPT. With it, you can ask your AI assistant questions about your data and run actions without leaving the chat.

- [Free connection](/mcp-and-integrations/free-mcp) - playbooks and KPI definitions, no account needed.
- [Paid connection](/mcp-and-integrations/paid-mcp) - read and write your live DigitalCore data.
- [API keys](/mcp-and-integrations/api-keys) - generate and manage keys for the paid connection.

## Data import
