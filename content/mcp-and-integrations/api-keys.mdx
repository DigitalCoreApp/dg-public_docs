# API Keys

API keys let an AI assistant or external tool access your DigitalCore data on your behalf.

## Create a key

1. Open **Account**, then **API Keys**.
2. Click **New Key**.
3. Give it a name ("Claude Desktop", "Internal automation").
4. Pick the access level: read-only, read and write, or admin.
5. Set an expiry (optional).
6. Click **Create**.

The key is shown once. Copy it and store it somewhere safe (1Password, secret manager). DigitalCore does not show it again.

## Use a key

Paste the key into the AI assistant connection setup. See [Paid MCP Setup](/mcp-and-integrations/paid-mcp) for client-specific instructions.

## Manage keys

The list shows each key''s name, access level, last used time, and expiry.

- **Rotate.** Create a new key, update the client, then revoke the old one.
- **Revoke.** Click the row, then **Revoke**. The key stops working immediately.

## Tips

- One key per client. Easier to revoke if a laptop is lost.
- Use read-only keys where possible.
- Set an expiry on shared keys.
