# Email Notifications

Email is the default channel for scheduled reports and personal alerts. You can also create shared distribution lists so reports go to a named group instead of a list of individual addresses.

## Personal alerts (per user)

In **Profile Settings** each user picks what they receive:

- Open Decisions assigned to them.
- Missed SLA terms on engagements they own.
- Mentions in comments.
- A daily or weekly digest.

## Distribution lists

Distribution lists are reusable recipient groups. Use one list per audience (for example "Exec Committee", "Customer A leadership", "Ops weekly").

### Create a list

1. Go to **Configure**, then **Notifications**, then **Email lists**.
2. Click **New list**.
3. Give it a name and an owner. The owner manages members.
4. Add members. Each member is just an email address. They do not need a DigitalCore account.

### Use a list

When you schedule a report or set up a signal alert:

1. In the recipients field, start typing the list name.
2. Pick the list. It expands to all current members at send time.
3. Add individual addresses alongside the list if needed.

### Manage subscribers

- The list owner can add or remove members at any time.
- Members can opt out using the **Unsubscribe** link in any email. They are removed from the list automatically.
- Set a **fallback recipient** on the list (usually the engagement owner). If the list is empty, the fallback receives the email.

## Scheduled reports (per report)

When you set up a report:

1. Open the report in **Communicate**, then **Reports**.
2. Click **Schedule**.
3. Pick monthly or quarterly.
4. Add recipients: distribution lists, individual addresses, or both.

External recipients do not need a DigitalCore account. They get the PDF and a public link.

## Sender and frequency

- Emails come from `notifications@digitalcore.app`. Add this to your safe senders list.
- Personal alerts can be real time, daily digest (8am local), or weekly digest (Monday morning).
- Scheduled reports are sent at the time you pick.

## Tips

- For external recipients, send the public link rather than the PDF, so the report stays current.
- Use the daily digest if you find real-time alerts noisy.
- Audit lists every quarter. Remove people who have left.
