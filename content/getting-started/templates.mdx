# Configure Your Templates

A template is a reusable list of items you track. Build one template per domain, then attach it to engagements.

## Finance template

Go to **Configure → Templates → Finance**. Click **New Template** and add items from the finance library.

Finance items group into:

- **Revenue**: recurring fees, project fees, ad-hoc charges.
- **Direct cost**: labour, subcontractors, materials.
- **Operating cost**: overhead, tooling, travel.

Each item has a planned and actual value per month. Labour items fill in automatically from capacity hours.

## Performance template

Go to **Configure → Templates → Performance**. Add KPIs from the performance library: First Call Resolution, Response Time, Customer Satisfaction, Uptime.

For each KPI, set:

| Field | What it does |
|-------|--------------|
| Target | What good looks like. |
| Amber band | Warning range. |
| Red band | Below this is a miss. |
| Direction | Higher is better, lower is better, or target is best. |

## Capacity template

Go to **Configure → Templates → Capacity**. Add roles from the roles library: Senior Consultant, Service Desk Analyst, Project Manager. Each role carries an hourly rate.

Set planned hours per role per month. During check-in, planned versus actual hours feeds utilisation, FTE, and labour cost.

## Attach templates to an engagement

Open the engagement and pick the three templates. Each template item creates an engagement-level row with monthly targets.

Different engagements can use different templates. Defaults inherit from the template, and you can override per engagement.
