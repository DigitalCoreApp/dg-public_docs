# Engagement Overview

The Overview is your operational worklist. Every engagement in one sortable table with the headline numbers.

## Columns

| Column | Meaning |
|--------|---------|
| Name | Engagement name. Click to open the detail view. |
| Service | The [service](/platform/configure/service-setup/services) it belongs to. |
| Customer | The consuming party. |
| Status | Active, On hold, or Closed. |
| Health | Composite score with health colour. |
| Margin % | Current period margin. |
| Utilisation | Actual hours against planned hours. |
| Open risks | Count broken down by severity. |
| Owner | The person accountable. |
| Last activity | The most recent recording, signal, or decision. |

Sort and filter by any column. The default sort is health from worst to best so problems show first.

## Filters

- **Status** to switch between Active and All.
- **Portfolio or tag** to scope to a strategic grouping.
- **Owner** to see your engagements only.
- **Customer** for account reviews.

## Quick actions

Hover a row to:

- Open the detail page.
- Record hours.
- Record a finance actual.
- Record a KPI value.
- Open a [decision case](/platform/decisions/cases).
